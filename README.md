# Boldness_WorkingMemory_Zebrafish

Reproducible statistical analysis workflow for behavioral and cognitive phenotyping in zebrafish translational neuroscience research.

This repository contains datasets and R scripts associated with the manuscript:

"Individual differences in boldness influence working memory and stress-induced repetitive behaviors in zebrafish"

The project investigates relationships between boldness-related behavioral traits, cognitive performance, and repetitive behavioral phenotypes using quantitative behavioral analysis and statistical modeling approaches.
